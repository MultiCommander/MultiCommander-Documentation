Upgrading

Update Tool

When Multi Commander starts, it automatically checks for new versions. If an update is available, you will be prompted to upgrade. You can also check manually by selecting Menu > Help > Check for Updates or launch the update tool directly via Menu > Help > Launch MultiUpdate.

You can manually start the update tool by running MultiUpdate.exe from the installation folder of Multi Commander.

The update tool downloads the correct update package and replaces the necessary files. This process works for both installed and portable versions.

Make sure to close all running instances of Multi Commander before running the update tool. It cannot replace files that are in use. If Multi Commander is minimized to the system tray, closing the window using the 'X' button does not fully close it. To exit completely, right-click the tray icon and choose Close Multi Commander.

The update tool can also (optionally) download beta versions. If you encounter issues, you can access advanced options within the tool.
For more information, refer to the MultiUpdate documentation.

Upgrade Using Installer

You can also upgrade by downloading and running the installer package. If the installer detects an existing installation of Multi Commander, it will offer two options:
Upgrade installation or Uninstall installation.

Selecting Upgrade will replace all the necessary files with updated versions, ensuring your installation is up to date.
