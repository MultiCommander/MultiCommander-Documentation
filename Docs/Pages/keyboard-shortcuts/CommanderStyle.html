Default Keyboard Shortcuts (Commander-Style)

Multi Commander's Commander-Style keyboard layout is inspired by the classic Norton Commander interface, optimized for power users who prefer keyboard-driven file management. This comprehensive reference covers all default shortcuts organized by functionality and scope.

Keyboard Layout Note

These shortcuts represent the Commander-Style layout. Multi Commander also supports Windows Explorer-style shortcuts through the Quick Setup option. You can fully customize all keyboard shortcuts through Configuration → Keyboard Customization.

Keyboard Customization

Multi Commander offers extensive keyboard customization options to match your workflow preferences.

  • Create custom key combinations
  • Export/import key configurations
  • Quick setup presets available
  • Context-sensitive key bindings
Quick Setup Options
  • Commander-Style: Norton Commander inspired
  • Explorer-Style: Windows Explorer familiar
  • Custom: Create your own layout

Access: Configuration → Keyboard Customization

Customization Tip

You can export your customized keyboard layout to a text file for backup or sharing with other users. This makes it easy to maintain consistent shortcuts across multiple installations.

Keyboard Layout Legend

Understanding the keyboard notation used in this documentation:

Key Notation
Ctrl Control key
Alt Alternate key
Shift Shift key
Num Numeric keypad
F1-F12 Function keys
Combination Keys

Key combinations are shown with + between keys:

  • Ctrl + C means press Ctrl and C together
  • Shift + F5 means press Shift and F5 together
  • Alt + Enter means press Alt and Enter together

Note: Press and hold the modifier keys (Ctrl, Alt, Shift) while pressing the main key.
