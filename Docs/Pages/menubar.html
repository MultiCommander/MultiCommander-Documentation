Menu Bar

Multi Commander's Menu Bar serves as the primary navigation hub for the application, providing comprehensive access to all major functions, features, and configuration options through a traditional menu system. This essential interface component organizes Multi Commander's extensive functionality into logical categories, offering both novice and advanced users intuitive access to file operations, system configuration, and specialized tools while supporting full customization through user-defined commands and menu structures.

Comprehensive Command Center

Access Multi Commander's complete feature set through a logically organized menu system. The Menu Bar provides hierarchical access to file operations, editing tools, view controls, configuration options, extensions, utilities, and help resources, all designed to enhance productivity while maintaining familiar interface patterns.

Menu Bar Layout and Customization

The Menu Bar provides structured access to Multi Commander's complete functionality through seven default menu categories, with full support for custom menu creation and user-defined command integration.

Menu Structure and Organization

Menu Bar showing the seven default menu categories for comprehensive application control

Default Menu Categories
  • File: File operations, tab management, and application control
  • Edit: Clipboard operations, selection tools, and folder comparison functions
  • View: Display controls, toolbar management, and layout options
  • Configuration: Settings, customization, and system configuration
  • Extensions: Plugin management and specialized tools
  • Tools: Utility functions, file processing, and system tools
  • Help: Documentation, support, and application information
Visibility and Customization Options
Menu Bar Control
  • Toggle Visibility: Ctrl+M to show/hide the Menu Bar
  • Space Optimization: Hide menu bar to maximize workspace area
  • Keyboard Access: Alt key activates menu navigation
  • Context Sensitivity: Menu items enable/disable based on current state
Customization Features
  • User-Defined Commands: Add custom commands to existing menus
  • Custom Menus: Create entirely new menu categories
  • Menu Organization: Organize commands with separators and hierarchies
  • Command Integration: Full integration with User Defined Commands

File Menu

The File menu provides comprehensive access to file operations, window management, and application control, serving as the primary entry point for most file-related tasks and system operations.

File Operations and Management

File menu displaying comprehensive file operations and application management options

Core File Operations
  • New: Create new Multi Commander windows, administrator windows, or Explorer Panel tabs
  • Tab Management: Close, duplicate, move, and lock tabs with advanced tab operations
  • View/Edit File: Open files with associated viewers or editors
  • Copy/Move/Delete: Access to comprehensive file operation dialogs
  • Archive Operations: Pack, unpack, and test archive files
  • Find Files: Launch the file search system
Key Features
  • Copy (Quick): Instant backup creation with .bak extension (Shift+F5)
  • Archive Testing: Verify integrity of compressed files
  • Window Management: Create new instances with different privilege levels
  • Tab Operations: Advanced tab manipulation and organization tools

Edit Menu

The Edit menu encompasses clipboard operations, advanced selection tools, and powerful folder comparison functions, providing sophisticated content management capabilities for efficient file organization and analysis.

Selection and Comparison Tools

Edit menu featuring clipboard operations, selection management, and advanced folder comparison tools

Clipboard and Selection Management
  • Standard Operations: Cut, Copy, Paste, and Delete with context awareness
  • Selection Tools: Expand, shrink, invert, and manage file/folder selections
  • Pattern Selection: Select by extension, name, or custom criteria
  • Selection Persistence: Save/load selection lists to memory or files
Advanced Folder Comparison
  • Newest/Missing: Identify files that are newer or missing between panels
  • Duplicate Detection: Find identical files across directories
  • Synchronization Support: Tools for maintaining folder consistency
  • Advanced Comparison: Multi-criteria comparison with customizable profiles

View Menu

The View menu controls Multi Commander's interface layout, toolbar visibility, and display options, enabling users to customize their workspace for optimal productivity and screen space utilization.

Interface and Layout Control

View menu providing comprehensive interface customization and layout management options

Layout and Panel Management
  • Split Size: Configure panel ratios (0/100, 25/75, 50/50, 75/25, 100/0)
  • Full Pane Width: Maximize active panel (Ctrl+F11)
  • Vertical Layout: Toggle between vertical and horizontal panel splits
  • Explorer Panel Modes: Switch between Details, List, and Thumbnail views
Interface Component Control
  • Toolbar Management: Show/hide Command Line Bar, Standard Commands, File Commands, and Drive Bar
  • Window Controls: Toggle Log window (Ctrl+L), Button window (Ctrl+B), and Menu bar (Ctrl+M)
  • Display Options: Control date formats and view modes
  • System Operations: Refresh (F2/F5) and Stop (Esc) functions

Tools Menu

The Tools menu offers an extensive collection of utility functions, file processing tools, and system operations, providing comprehensive solutions for file management, content manipulation, and system administration tasks.

Comprehensive Utility Collection

Tools menu featuring extensive utility functions and specialized processing tools

File and Folder Operations
  • Folder Management: Calculate folder sizes, create folders, and correct date/time
  • File Properties: Change attributes, timestamps, and find locked files
  • Clipboard Tools: Export paths, names, dates, and content in various formats
  • File Links: Create symbolic links and junctions
Content Processing Tools
  • Text Tools: Format conversion (Unix/Windows/Mac), XML processing, line sorting, find and replace
  • Encoding/Decoding: Base64 encoding and decoding operations
  • Rename Tools: Case conversion, text replacement, and extension management
  • Media Tools: Picture processing, audio metadata, and video analysis
System and Security Tools
  • Windows Integration: Direct access to Windows Control Panel applets
  • File Security: Permission management, ownership control (Admin required)
  • Special Operations: Duplicate selection by version and advanced file processing

Menu Bar Mastery

Master the Menu Bar by understanding its logical organization: use File menu for operations and window management, Edit menu for selection and comparison, View menu for interface control, Configuration menu for customization, Extensions menu for specialized tools, Tools menu for utilities and processing, and Help menu for support and maintenance. Remember that Ctrl+M toggles visibility, and the menu system fully integrates with user-defined commands for complete workflow customization.
