Text Tools

Under the Tools menu you can find some special Text Tools.

There are tools for converting the line endings of text files to those different platforms and a tool for converting tab characters to spaces.

And you can also reformat XML files to structured layout or compact layout.

You can find more information about them under Text Conversion Tools.

Sort Lines will sort the lines in text files using many advanced options.

Find and Replace will allow you find find and replace text in files. It support MultiLine find/replace matches and many other options.
