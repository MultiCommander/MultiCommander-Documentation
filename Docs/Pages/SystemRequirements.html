System Requirements

Multi Commander works on most operating systems from Windows XP and Windows Server 2003 and newer.

Client Operating System : Windows 11, Windows 10, Windows 8.1, Windows 8, Windows 7, Windows Vista , Windows XP*, both 32bit and 64bit
Server Operating System : Windows Server 2016, Windows Server 2012/R2, Windows Server 2008/R2, Windows Server 2003, both 32bit and 64bit
Not all features are available on all operating systems. Note that from summer 2016, XP support is not support in latest version. But latest version supporting XP is available for download.

Minimal Hardware : 32bit CPU with SSE2 support
Screen Resolution : A minimum resolution of 800x600 is recommended
